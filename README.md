# Mini-React

一个简易版的 React 框架，用于说明 React 的实现原理。

# 实现核心功能

实现了 React 的 JSX，虚拟 DOM，组件，生命周期，diff 算法，异步的 setState 等核心功能。

# 运行

> 1. git clone <https://github.com/xingpengchao/Mini-React.git>
> 2. cd Mini-React
> 3. npm install
> 4. npm start
> 5. 浏览器打开 localhost:1234
