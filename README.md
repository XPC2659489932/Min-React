# Mini-React
一个简易版的React框架，用于说明React的实现原理。

# 实现核心功能

实现了React的JSX，虚拟DOM，组件，生命周期，diff算法，异步的setState等核心功能。

# 运行

1. git clone https://github.com/xingpengchao/Mini-React.git
2. npm install
3. cd Mini-React
4. npm start
5. 浏览器打开localhost:1234
